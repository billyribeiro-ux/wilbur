# Whiteboard Keyboard Shortcuts

## General Actions

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + Z` | Undo last action |
| `Cmd/Ctrl + Shift + Z` or `Ctrl + Y` | Redo action |
| `Esc` | Cancel current action / Deselect |

## Tool Selection

| Shortcut | Tool |
|----------|------|
| `V` | Select tool |
| `H` | Hand tool (pan) |
| `P` | Pen tool |
| `E` | Eraser tool |
| `T` | Text tool |
| `L` | Line tool |
| `R` | Rectangle tool |
| `C` | Circle tool |
| `A` | Arrow tool |

## Text Formatting

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + B` | Toggle bold |
| `Cmd/Ctrl + =` | Increase font size |
| `Cmd/Ctrl + -` | Decrease font size |
| `Enter` | New line in text box |
| `Cmd/Ctrl + Enter` | Create new text box |

## Text Editing

| Shortcut | Action |
|----------|--------|
| `Esc` | Cancel text editing |
| `Enter` | Commit text (when editing) |
| `Cmd/Ctrl + Enter` | Finish editing and create new text box |

## Navigation (Future)

| Shortcut | Action |
|----------|--------|
| `Arrow Keys` | Move selected object (1px) |
| `Shift + Arrow Keys` | Move selected object (10px) |

---

**Note:** All shortcuts work when the whiteboard is active. Text-specific shortcuts only work when the text tool is selected or when editing text.
